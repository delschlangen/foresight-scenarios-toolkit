Foresight Methodology

A structured approach to strategic foresight, adapted from Shell's scenario planning methodology.

1

Identify Drivers

Map 10-15 forces shaping the future using STEEP+T framework

  • Social, Technological, Economic
  • Environmental, Political, Threat
→
2

Select Uncertainties

Choose 2 critical uncertainties as scenario axes

  • High impact on focal question
  • High uncertainty
  • Independent of each other
→
3

Build Scenarios

Develop 4 narratives for each quadrant

  • Plausible, not predictions
  • Internally consistent
  • Meaningfully different
→
4

Define Indicators

Create early warning system

  • Leading indicators
  • Confirming indicators
  • Lagging indicators
→
5

Operationalize

Connect to decisions

  • Weekly horizon scans
  • Quarterly reviews
  • Strategy stress-tests

Key Principles

✓

Plausible

Scenarios could actually happen given current trends and dynamics

✓

Differentiated

Each scenario represents a meaningfully different world

✓

Challenging

Scenarios force new thinking and stress-test assumptions

✓

Decision-relevant

Scenarios connect directly to strategic choices

Avoid These Pitfalls

✗

"Good future vs. Bad future" — Not insightful; each scenario should have winners and losers

✗

Predictions — Scenarios are possibilities, not forecasts

✗

Science fiction — Plausibility matters; ground scenarios in reality
